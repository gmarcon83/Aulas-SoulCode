Tamanhos

    .form-control-sm => diminui o tamanho do input;
    .form-control-lg => aumenta o tamanho do input;
    .col-form-label-sm => diminui o tamanho da label;
    .col-form-label-lg => aumenta o tamanho da label;


Input Group

    .input-group => agrupa os elementos filhos em um grupo
    .input-group-prepend => adiciona o elemento como inicio do grupo
    .input-group-append => adiciona o elemento como fim do grupo
    .input-group-text => forma o texto como elemento do grupo


Checkbox e Radio

    .form-check => no elemento pai;
    .form-check-inline => adicionado ao de cima para tornar inline;
    .form-check-input => no input;
    .form-check-label => no label;


Checkbox e Radio Custom

    .custom-control+custom-checkbox => no elemento pai
    .custom-control-input => no input;
    .custom-control-label => no label;


Elementos custom

    .custom-select => estilo alternativo para selects;
    .custom-range => estilo alternativo para ranges;
