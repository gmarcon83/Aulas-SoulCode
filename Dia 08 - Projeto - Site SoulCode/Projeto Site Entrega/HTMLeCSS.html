HTML e CSS

A Web é a plataforma de programação mais importante do mundo, e é praticamente onipresente. Suas tecnologias principais, o HTML e o CSS, são fundamentais para a criação de qualquer site. Quem domina essas tecnologias, suas boas práticas e seus recursos mais avançados, tem espaço no mercado de programação para front-end e pode participar da criação de plataformas incríveis. A formação HTML e CSS foi pensada para você começar do zero e se tornar um profissional de ponta.


    Você aprenderá:

  • Criação e composição de ilustrações vetoriais
  • Diagramação e edição
  • Criação de peças gráficas completas
  • Conteúdo específico de ferramentas e funções
  • Bitmaps, tipografia e efeito
