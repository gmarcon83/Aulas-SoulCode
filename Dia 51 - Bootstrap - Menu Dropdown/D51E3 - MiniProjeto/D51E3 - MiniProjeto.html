Bootstrap

Bootstrap é um framework de front end usado para facilitar o desenvolvimento de sites responsivos.

O Bootstrap inclui templates HTML e CSS usados para estilizar componentes comuns em páginas web, entre eles podemos destacar: botões, campos de texto, dropdowns, cores, bordas, alinhamento, margens, estilos de texto, entre outros

Sendo desenvolvido inicialmente pelo Twitter em 2010, o Bootstrap foi disponibilizado em um modelo de framework open-source logo em seguida.

Grid

O sistema de grid do bootstrap é uma maneira rápida de criar páginas responsivas.

A grid do Bootstrap é dividida em 12 colunas que mudam de tamanho de acordo com a resolução do dispositivo.

Pra usar as grids no Bootstrap adicionamos a classe "row"no elemento pai e adicionamos as classes "col" nos elementos filhos.

Na classe "col" podemos opcionalmente adicionar no final o numero de colunas que queremos que o elemento ocupe e no meio a partir de que breakpoint essa configuração começa a fazer efeito. "col-md-6" por exemplo indicaria que queremos que o elemento ocupe 6 colunas a partir do breakpoint medio.

Exemplo

Uma coluna com largura de 4 e uma de 8, que quebram no breakpoint médio.

col-md-4
col-md-8

Cores

Para ajudar a criar uma consistencia visual no site o Bootstrap ja vem com uma pré-seleção de cores para serem usadas.

As cores no Bootstrap são adicionadas no fim das classes dos elementos para alterar as cores dos mesmos. Em um elemento "btn" por exemplo adicionamos uma classe "btn-nomedacor" onde nome da cor pode ser: "primary", "secondary", "success", "info", "warning", "danger" e "muted".

Bordas

O Bootstrap conta também com classes para adicionar bordas nos diversos elementos do site de maneira padronizada.

Para adicionar bordas nos elementos usando o Bootstrap usamos a classe "border". Podemos adicionar "-top", "-right", "-bottom" e "-left" para adicionar bordas apenas em um dos lados do elemento.

Para remover as bordas usamos a mesma sintaxe de adicionar mas com um "-0" no final. Por exemplo "border-top-0".

Para adicionar cores nas bordas adicionamos a classe "border-nomedacor", onde nomedacor é uma das cores faladas anteriormente. Por exemplo "border-danger".

Exemplos

border-right-0
border-danger
border-top

Tipografia

A tipografia do Bootstrap suporta estilizar e formatar os elementos de texto em headings, paragrafos, citações, entre outras.

As fontes no Bootstrap tem um tamanho padrão de 1rem e uma altura de linha de 1.5. Adicionalmente é aplicada uma margem em baixo de todos os parágrafos em uma altura de metade da altura das linhas.

Os comandos de texto são adicionados com classes que iniciam em "text". "text-left", "text-center", "text-right" e "text-justify" por exemplo ajustam o alinhamento do texto.

Adicionando "lowercase", "uppercase" ou "capitalize" alteramos as letras para minúsculas, maiúsculas e primeira letra maiúscula, respectivamente.

Usando as classes "font-weight-bold", "font-weight-normal" e "font-weight-light" deixamos o texto em negrito, normal ou leve.

Podemos adicinar as classes de cores do Bootstrap também para alterar a cor do texto. Por exemplo "text-danger" e "text-muted".

Margens e Paddings

Podemos adicionar paddings e margens nos elementos com o Bootstrap de maneira padronizada usando as algumas de suas classes.

Para adicionar margens com o Bootstrap usando m, para margem e p para padding. A isso podemos adicionar t para topo, b para baixo, l para esquerda, d para direita, x para horizontal e y para vertical. Os valores numéricos vão de -0 até -5

Exemplo: "mt-4" adicionaria uma margem no topo com espaçamento 4.

Botões e Dropdowns

Criar botões e dropdowns é muito prático usado as classes do Bootstrap.

A classe básica para criar um botão é a classe "btn". Adicionando "btn-nomedacor", onde nomedacor é uma das cores faladas anteriormente, escolhemos as cores dos botões.

Para adicionarmos dropdowns usamos no botão a classe "dropdown-toggle" e o atributo data-toggle="dropdown". Na div com os itens usamos a classe "dropdown-menu" e nos itens do dropdown usamos "dropdown-item".

Exemplos

btn btn-primary

Campos de Input

Para auxiliar na criação de formulários o Bootstrap conta com classes também para os campos de input.

Nos campos de input adicionamos a classe "form-control" para formatar o campo com display: block e width: 100%, além de ajustes na estilização do elemento.

Como opção de layout de formulários podemos usar eles dentro das grids do Bootstrap, onde existe a classe "form-row" para usar no lugar de "row", diminuindo o gap entre os elementos.

Podemos agrupar o campo com sua label colocando os dois dentro de uma div com a classe "form-group". Se preferirmos os dois em linha usamos a classe "form-inline" nos elementos.

Existe também a opção de usar "input-group" para agrupar elementos com os inputs, criando efeitos de continuidade com outros elementos, como textos e botões. A estes adicionamos "input-group-append" e "input-group-prepend" se eles aparecem no inicio ou no fim do grupo, respectivamente.
